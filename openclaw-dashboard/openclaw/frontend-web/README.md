# OpenClaw Dashboard Channel: Frontend (React)

## Structure
- Page for each key OpenClaw capability (Chat, Agents, Skills, CronJobs, Sessions)
- Uses Material UI for layout
- API client stubs to be implemented in `/src/api/`
- Each feature starts as a page/component stub for fast extension

## Getting Started
1. `cd frontend-web`
2. `npm install`
3. `npm start`
